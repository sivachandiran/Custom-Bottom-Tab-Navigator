# Installation


using npm:

```
npm i @react-navigation/native
npm install @react-navigation/bottom-tabs
```

using yarn:

```
yarn add @react-navigation/native
yarn add @react-navigation/bottom-tabs
```

### Supported platform

- Android
- Ios
